# MVP-38 Final Review — Audience Paths

## Founder Path

- **Focus:** strategic narrative, positioning, conversion story
- **Duration:** 2 minutes
- **Flow:** Entry Gate → Demo Script Overview → Walkthrough Path (high-level) → Copy Tone Check → Checklist Executive Summary → Sign-off

### Founder Path Instructions
1. Read the demo overview section in the script lock file
2. Walk through the high-level path without clicking — verify the story arc
3. Review the copy bank for tone and brand alignment
4. Read the executive summary row in the checklist
5. Mark founder sign-off in the checklist
6. Flag any strategic concerns to operator before final gate

## Recruiter Path

- **Focus:** role accuracy, team scope, candidate-facing interaction patterns
- **Duration:** 3 minutes
- **Flow:** Entry Gate → Demo Path (candidate perspective) → Audience Path Review → Recruiter Checklist Section → Sign-off

### Recruiter Path Instructions
1. Read the demo script with a candidate-interaction lens
2. Verify the walkthrough path includes realistic user flows
3. Review the recruiter-specific row in audience paths
4. Confirm handoff points are accurate for team workflow
5. Mark recruiter sign-off in the checklist
6. Flag any mismatches between script and actual team process

## Technical Reviewer Path

- **Focus:** technical accuracy, edge cases, system behavior, error handling
- **Duration:** 4 minutes
- **Flow:** Entry Gate → Demo Script (detailed audit) → Walkthrough Steps (click-by-click) → Edge Case Review → Technical Checklist Section → Sign-off

### Technical Reviewer Path Instructions
1. Audit every click-by-click instruction against actual system behavior
2. Verify error states, validation feedback, and fallback handling
3. Check that the walkthrough path covers all critical system interactions
4. Confirm timing estimates are feasible given system response times
5. Mark technical reviewer sign-off in the checklist
6. Document any technical discrepancies and block sign-off until resolved

## Operator Path

- **Focus:** final gate, consolidated review, go/no-go decision
- **Duration:** 3 minutes
- **Flow:** Entry Gate → Verify All Files Present → Review All Sign-offs → Validate Checklist Completion → Make Go/No-Go Call → Record Decision

### Operator Path Instructions
1. Confirm all seven export files exist and the manifest is accurate
2. Verify that each audience section sign-off is marked in the checklist
3. Read the consolidated decision summary in the checklist
4. Validate timing estimates sum correctly
5. Make the go/no-go call
6. Record the decision with rationale and timestamp
7. If no-go, document exact blockers and required remediation scope
