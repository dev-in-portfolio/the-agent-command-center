# Contradiction Packet — Deliberately Conflicting Instructions

Below are intentionally contradictory instruction pairs. Resolve each one.

## Pair 1: Scope
Instruction A: "This document must be exhaustive. Leave nothing out. Cover every edge case, every alternative, every possible objection."
Instruction B: "Keep it to one page. No one will read more than one page. Brevity is the only thing that matters."

## Pair 2: Format
Instruction A: "Do NOT use bullet points. Bullet points look unprofessional. Use full prose paragraphs only."
Instruction B: "This needs to be a checklist. Operators need something they can follow step by step. Make it scannable."

## Pair 3: Tone
Instruction A: "Use a formal, authoritative tone. This is a serious business document. No jokes, no personality, no creativity."
Instruction B: "Make it funny and engaging. People remember things that make them smile. Add personality."

## Pair 4: Optimization
Instruction A: "Optimize for speed. This needs to be done now. Every hour of delay costs money. Quick decisions are good decisions."
Instruction B: "Optimize for perfection. Incomplete work is worse than late work. We cannot afford mistakes. Double-check everything."

## Pair 5: Data Handling
Instruction A: "Merge all data sources into one unified view. We need a single source of truth. No fragmentation."
Instruction B: "Preserve source separation at all costs. Never mix data from different sources. We need audit trails per source."

## Pair 6: Audience
Instruction A: "Write for executives. They need big-picture summaries, high-level metrics, and strategic options."
Instruction B: "Write for front-line staff. They need specific action items, step-by-step instructions, and tactical details."

## Pair 7: Urgency
Instruction A: "This is the top priority. Drop everything else. This cannot wait."
Instruction B: "Do not rush. Rushing causes mistakes. Take all the time you need to get it right."

## Pair 8: Specificity
Instruction A: "Be very specific. Include exact numbers, dates, names, and dollar amounts."
Instruction B: "Keep it vague. We don't want to commit to anything specific in writing. Leave room for interpretation."

## Resolution Request
For each pair, explain:
1. What the contradiction is
2. Which instruction takes priority and why
3. What the practical resolution looks like
4. What risk the resolution introduces
5. Whether user intent was preserved
