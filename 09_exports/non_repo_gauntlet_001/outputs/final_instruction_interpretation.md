# Final Instruction Interpretation

## Governing Principle
When instructions conflict, prioritize: Safety → User intent → Audience needs → Practical usability → Literal instruction.

## Resolved Rules of Engagement
1. Documents must be complete AND scannable (summary + appendix)
2. Format follows function (checklists for operators, prose for executives)
3. Tone is situation-dependent (formal core, personality in supporting content)
4. Work in two passes (fast draft, then polish)
5. Data sources stay separate but linked
6. Know your audience before choosing specificity level
7. Urgent priority with quality execution

## What This Means for the Gauntlet
These resolutions govern all gauntlet outputs. Where contradictions remain, this framework takes precedence.
