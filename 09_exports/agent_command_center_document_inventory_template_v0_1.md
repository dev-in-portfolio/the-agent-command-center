# Agent Command Center Document Inventory Template v0.1

## Current Context
- Station Chief runtime is parked at v4.7.0.
- This is a non-runtime document inventory template.
- This document does not modify runtime behavior.
- This document does not create v4.8.
- This document does not authorize runtime behavior.

## Purpose
This template provides a structured inventory format for non-runtime documentation files, their purpose, status, category, and runtime effect.

- this is a template only
- it does not inspect files automatically
- it does not modify existing documents
- it does not select future work
- it does not grant permissions
- it does not authorize v4.8

## Document Inventory Principle
- inventory entries are references only
- inventory entries do not modify documents
- inventory entries do not imply task priority
- inventory entries do not grant permissions
- inventory entries do not create runtime behavior
- inventory entries do not create v4.8
- inventory entries do not select next tasks

## Inventory Fields

- **Document name**: Descriptive name.
- **File path**: Repository path.
- **Category**: Documentation family.
- **Version**: Document version (v0.1).
- **Status**: Present, Drafted, Landed, Parked.
- **Purpose**: One-sentence explanation.
- **Runtime effect**: None.
- **Authorizes future work**: No.
- **Related documents**: Linkages.
- **Operator notes**: Tracking notes.
- **Last known commit**: Hash.
- **Review status**: Pending/Reviewed.

## Document Category Definitions

- **architecture planning**: Mapping families/concepts. Effect: None.
- **permission planning**: Access/approval boundaries. Effect: None.
- **safety planning**: Risk/mitigation design. Effect: None.
- **operator governance**: Authority/protocol. Effect: None.
- **task taxonomy**: Classification/boundaries. Effect: None.
- **worker glossary**: Terminology/naming. Effect: None.
- **documentation index**: Inventory of file set. Effect: None.
- **prompt archive**: Index of prompt categories. Effect: None.
- **dashboard planning**: UI/UX/metric visibility. Effect: None.
- **handoff reporting**: Handoff/landing report format. Effect: None.
- **control register**: Parking/violation/audit records. Effect: None.
- **command language**: Command/interpretation definition. Effect: None.
- **mode control**: Switching logic/rules. Effect: None.
- **status vocabulary**: Definitions for reports/dashboards. Effect: None.
- **approval vocabulary**: Phrase registry. Effect: None.
- **scope template**: Prompt/task boundaries. Effect: None.
- **progress metric**: Metric definitions. Effect: None.
- **inventory template**: Reference management. Effect: None.
