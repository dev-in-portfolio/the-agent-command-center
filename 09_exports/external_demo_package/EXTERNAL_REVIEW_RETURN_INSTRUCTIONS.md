# External Review Return Instructions

To complete your review, please return your feedback packet using one of the following methods.

## Secure Return Policy
- **NO SECRETS:** Ensure no personal tokens or credentials are in your return.
- **NO CLIENT DATA:** Do not include any private project data in the feedback.

## Method 1: Dashboard Copy (Recommended)
1. Use the "Feedback Intake" tool on the live dashboard.
2. Fill out the fields.
3. Click "Generate Feedback Packet."
4. Copy the resulting text and send it via the agreed-upon channel.

## Method 2: Manual Markdown Return
1. Download the [REVIEWER_RESPONSE_FORM.md](./REVIEWER_RESPONSE_FORM.md).
2. Fill out the fields manually.
3. Return the completed markdown file.

Thank you again for your time!
