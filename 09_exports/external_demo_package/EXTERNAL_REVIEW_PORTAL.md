# External Review Portal — The Agent Command Center

This document serves as the top-level entry point for the external review process.

## Quickstart
1. Open the [START_HERE.md](./START_HERE.md) guide.
2. Select your persona in [ROLE_BASED_REVIEW_PATHS.md](./ROLE_BASED_REVIEW_PATHS.md).
3. Review the [Product One-Pager](./product_one_pager.md).

## Portal Navigation
- **Onboarding:** [START_HERE.md](./START_HERE.md), [REVIEW_PACKET_INDEX.md](./REVIEW_PACKET_INDEX.md)
- **Personas:** Technical, Recruiter, Founder, Collaborator.
- **Safety:** [Safety Boundary Brief](./safety_boundary_brief.md), [SHARE_SAFE_CHECKLIST.md](./SHARE_SAFE_CHECKLIST.md)

## Review Integrity
This portal and the associated demo package have been QA'd to ensure:
- Zero secrets disclosure.
- Zero persistent token storage.
- Non-hype, honest product positioning.
- Clearly blocked approval/execution/automation paths.
